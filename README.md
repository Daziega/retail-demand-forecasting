# Retail Demand Forecasting & Inventory Optimization

## Project Overview
This project builds a machine learning system to forecast retail demand
and optimize inventory decisions using historical sales, promotions,
calendar events, and external data.

## Objectives
- Improve demand forecasting accuracy
- Reduce stockouts and overstock
- Translate predictions into inventory decisions

## Methodology
CRISP-DM (Business Understanding → Deployment)

## Tech Stack
Python, Pandas, Scikit-learn, LightGBM, Streamlit
